# Billionaires Dashboard – Power BI Project

## 📌 Overview
The **Billionaires Dashboard** is an interactive Power BI project used to analyze billionaire data such as tax revenue, GDP growth, population, life expectancy, and categories across different countries.

---

## 📊 Features

### KPI Cards
- Total Tax Revenue
- Population
- GDP
- Life Expectancy
- Tax Rate

### Interactive Slicers
- Country
- Full Name
- Category
- Source
- Rank

### Visualizations
- Charts and tables for data analysis

---

## 🛠️ Tools Used
- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

---

## 📈 Insights
- Male billionaires contribute more tax revenue.
- UK and USA show high GDP growth.
- Automotive is a major billionaire category.

---

## 🚀 How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the dataset if needed.
3. Use filters to interact with the dashboard.
